# DESIGN

## Features
- **Daily Digest**: Summarize and send NSE corporate announcements.
- **XBRL Parsing**: Extract specific elements from raw XBRL XML documents.
- **Market Enrichment**: Add stock price, market cap, and 52-week high/low data.
- **Industry Mapping**: Map symbols to their corresponding industry sectors.
- **Discord Delivery**: Format the enriched data and send it as rich embeds to Discord.
