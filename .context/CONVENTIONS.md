# CONVENTIONS

This project adheres to the following coding standards:

## Standards
- **Linter & Formatter**: We use `ruff` for linting and formatting. Run using:
  ```bash
  uv run ruff check . --fix
  uv run ruff format .
  ```
- **Type Hints**: All function signatures must have type hints.
- **Docstrings**: Public functions, classes, and methods require Google Style docstrings.
- **Module Docstrings**: Every `.py` file must start with a docstring explaining its purpose.
