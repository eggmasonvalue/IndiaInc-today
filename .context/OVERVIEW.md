# OVERVIEW

## Purpose
IndiaInc Today is a daily digest system that harvests key corporate XBRL announcements from the National Stock Exchange (NSE), enriches them with current market data and industry mapping, and delivers a polished digest to a Discord webhook.

## Scope
- Focuses on 8 key announcement types defined in XBRL filings (Reg30, fundRaising, agr, award, annFraud, cdr, CIRP, annOts).
- Currently NSE-only (Mainboard & SME). BSE support is deferred for future phases.
- Executed daily via GitHub Actions.
